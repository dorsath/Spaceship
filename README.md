# Spaceship shooter

A sunday morning experiment in opengl-ruby

## Installation

```
./install
```

## Usage

```
./start
```

## Controls

* `W`: turn camera up
* `S`: turn camera down
* `A`: turn camera to the left
* `D`: turn camera to the right

* `f`: fullscreen
* `F`: exit fullscreen
