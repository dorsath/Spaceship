# OpenGL in Ruby

Just some things we made on an evening of happy hacking.

## Installation

```
./install
```

## Usage

```
./start
```

## Controls

* `W`: jump
* `A`: move left
* `D`: move right
* `space`: shoot
* `R`: reload

* `f`: fullscreen
* `F`: exit fullscreen

* `h`: move screen to left
* `l`: move screen to right
